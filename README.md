# האתר של ארבעת הציידים

מערכת וובית בעברית לניתוח שוק קריפטו וזהב באמצעות:

- 28 נכסים חיים הנסחרים מול USDT ב-Binance
- ניתוח רב־טווח במקביל: 15 דקות, שעה, 4 שעות ויום
- מבנה שוק, EMA 20/50/200, RSI, MACD, VWAP, ATR, ADX, Bollinger ו-Stochastic
- Fibonacci על טווח המסחר האחרון
- זיהוי היוריסטי של Order Blocks ו-Fair Value Gaps
- סריקות נזילות והקשר Wyckoff
- מנוע תרחישים עולה / דשדוש / יורד עם פירוט גורמי הציון
- תוכנית תרחיש מותנית עם אזור מעקב, ביטול תזה, יעדים ויחס סיכוי/סיכון
- לוח קונפלואנסים שקוף שמציג אילו אותות שוריים, דוביים או ניטרליים
- מסך כניסה ששומר הרשאה ברמת session עד לסגירת הלשונית
- מפת תרחישים לפי 24 שעות, 7 ימים ו-30 ימים
- יצוא PNG מקורי הכולל גרף, רמות מפתח, שלושה מסלולים עתידיים, הסתברויות וגורמי קונפלואנס

## הפעלה

מתוך תיקיית הפרויקט:

```powershell
python -m http.server 8080 --directory outputs/market-atlas
```

ואז לפתוח:

```text
http://localhost:8080
```

האפליקציה מנסה לקבל 300 נרות לכל אחד מארבעת טווחי הזמן מ-Binance Public API. אם החיבור הראשי נכשל היא עוברת אוטומטית לנתוני הדמיה דטרמיניסטיים ומציגה זאת בכותרת.

## הערה חשובה

ה"אחוזים" הם ניקוד תרחישים היוריסטי המבוסס על נתוני עבר. הם אינם הסתברות סטטיסטית מכוילת, אינם מבטיחים תוצאה ואינם ייעוץ או הוראת מסחר. אין במערכת ביצוע פקודות מסחר.

מסך הכניסה באתר הסטטי הוא נעילת ממשק בצד הלקוח. הוא אינו מחליף אימות שרת, ולכן אינו מתאים להגנה על מידע סודי או רגיש.
