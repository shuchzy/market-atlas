# Market Atlas

אב־טיפוס וובי בעברית לניתוח שוק קריפטו וזהב באמצעות:

- מבנה שוק, EMA 20/50/200, RSI, MACD, VWAP ו-ATR
- Fibonacci על טווח המסחר האחרון
- זיהוי היוריסטי של Order Blocks ו-Fair Value Gaps
- סריקות נזילות והקשר Wyckoff
- מנוע תרחישים עולה / דשדוש / יורד עם פירוט גורמי הציון

## הפעלה

מתוך תיקיית הפרויקט:

```powershell
python -m http.server 8080 --directory outputs/market-atlas
```

ואז לפתוח:

```text
http://localhost:8080
```

האפליקציה מנסה לקבל 300 נרות מ-Binance Public API. אם החיבור נכשל היא עוברת אוטומטית לנתוני הדמיה דטרמיניסטיים ומציגה זאת בכותרת.

## הערה חשובה

ה"אחוזים" הם ניקוד תרחישים היוריסטי המבוסס על נתוני עבר. הם אינם הסתברות סטטיסטית מכוילת, אינם מבטיחים תוצאה ואינם ייעוץ או הוראת מסחר. אין במערכת ביצוע פקודות מסחר.
